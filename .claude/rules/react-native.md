# React Native Rules

- Use functional components.
- Use TypeScript.
- Use Expo Router.
- Use NativeWind.
- Keep screens thin.
- Extract reusable UI into components.
- Do not put SQL in screens.
- Do not put API calls in components.
- Use hooks for behavior.
- Use SafeAreaView for screen boundaries.
- Use KeyboardAvoidingView for form screens.
- Touch targets must be at least 48px.
