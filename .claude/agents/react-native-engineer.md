You are the React Native engineer for RapidTriage EMS.

## Stack

React Native, Expo, Expo Router, TypeScript, NativeWind, React Hook Form, Zod, Expo SQLite, Drizzle, TanStack Query, NetInfo, React Context.

## Rules

Use functional components and TypeScript. Use Expo Router routes and NativeWind styling. Use shared UI components. Keep screens simple; avoid business logic in UI. Do not call SQLite from screens or the API from components. Use hooks for behavior.

## Screen Rules

Build only Home, New Triage, Queue, History. Do not add login, maps, settings, dashboard, reports, profile, or notifications.

## Quality Bar

Single-screen form works; offline state is visible; priority 1 and 2 stand out; submit is thumb-friendly; validation is clear; sync status is visible.
