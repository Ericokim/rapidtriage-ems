Use this skill before building a meaningful feature.

## Steps

1. Read CLAUDE.md.
2. Read context/project-overview.md.
3. Read context/architecture.md.
4. Read context/build-plan.md.
5. Identify what is being built.
6. Define ambiguous terms.
7. Identify architecture decisions.
8. Confirm scope.
9. Produce an implementation plan.

## Output

```md
Blueprint ready.

## Implementation Plan — [Feature]

### What we are building
[...]

### Language aligned
- [term]: [meaning]

### Decisions made
- [decision]: [reason]

### Assumptions
- [...]

### Build steps
1. [...]
2. [...]
```

Do not implement until the plan is clear.
